# Privacy Center - Bookmark Manager Ecosystem

A professional, high-impact **Neobrutalist "Acid Minimal"** Privacy Center built with Next.js 15. This hub hosts privacy policies for our browser extensions and web tools, starting with the **Zero Bookmark Manager**.

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Next.js](https://img.shields.io/badge/Next.js-15-black)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-v3-blue)

## Features

- **Neobrutalist Design**: Bold borders, high contrast, and "Acid Yellow" aesthetic.
- **Three Core Themes**: Light, Dark, and a default specialized Yellow theme.
- **Interactive AI Chatbot**: A local-processed mock AI to answer project-specific privacy queries.
- **Scroll Reveal Animations**: Smooth content transitions as the user scrolls.
- **Dynamic Policy Summary**: Instant summarization button for quick policy reading.
- **Background Effects**: Drifting dot patterns, grain/noise overlays, and floating geometric shapes.
- **Multi-language Support**: Fully localized in **English** and **Tiếng Việt**.

##  Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Themes**: Next Themes
- **Font**: Inter

## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ChickenSoup269/Zero-Bookmark-Manager.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Prepare for production**:
   ```bash
   npm run build
   ```
